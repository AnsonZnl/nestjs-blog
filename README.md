## 描述

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## 安装

```bash
$ npm install
```

## 启动

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
